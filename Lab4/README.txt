Lab 4: Syntax Checker


-----------
DESCRIPTION

In this lab, there is a file in which contains brackets and other ASCII values. It counts the amount of match brackets, while also making sure the are no mismatches. If there is a mismatch within the brackets and the stack an error will be printed and the program will be ended. If there are only matching brackets, the amount of brackets will be printed.

INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS). Enter the test case as a program argument and run using MARS. Click assemble and press the run button in order to find the mismatches or success within the file from the program argument.
