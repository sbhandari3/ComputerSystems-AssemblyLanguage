Lab 3: ASCII-risks (Asterisks)

-----------
DESCRIPTION

In this lab, the user enters a number which prints the number of rows for the triangle the triangle. The number must be above 0 and prompts the user to keep entering the number until it is greater than 0. Uses 3 loops to make sure the tabs and asterisks are printed correctly. 
-----------
FILES
Lab3.asm

This file includes the assembly code of the lab.

-----------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS). Enter the test case as a program argument and run using MARS. Click assemble and press the run button in order to be prompted to enter an integer.
