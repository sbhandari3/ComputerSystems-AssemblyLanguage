Lab 2: Simple Data Path

-----------
DESCRIPTION

In this lab, Registers are used in order to make ALU inputs which then determine what the ALU output is by rotating the first ALU input, the second ALU input amount of times. After the ALU output is produced the user should also be able to update the user's choice of register using the ALU output and the update button.

-----------
FILES

-
Lab2.lgi
This file includes the assembly code of the lab, from the app of Multimedia Logic otherwise known as MML.

-----------
INSTRUCTIONS

This program is intended to be run using Multimedia Logic otherwise known as MML. Using the switches provided you are to see whether the register provide the correct output and whether the ALU produces the right outputs as well.
